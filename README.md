# SAP Analytics Portfolio  
**SAP Analytics Cloud | SAP BW/4HANA | Integrated Planning**

Designed and implemented as hands-on system configurations in SAP Analytics Cloud and SAP BW/4HANA training environments to simulate real-world enterprise analytics and planning scenarios.

This repository contains selected SAP Analytics implementation projects demonstrating end-to-end dashboard development, planning configuration, data actions, allocations, KPI modeling, and BW live integration.

The implementations are based on a simulated mid-sized German manufacturing company and focus on delivering business-driven analytics and planning capabilities, including:

- Executive Sales Performance Reporting
- Contribution Margin Analytics
- Integrated Sales Planning & Forecasting
- Data Actions & Allocation Logic
- Version Management & Variance Analysis
- BW/4HANA Live Connectivity

--------

## Projects Included

### 1. Sales Performance Analytics (SAC + BW Live)
Interactive executive dashboard with KPI modeling, YoY variance, contribution margin analysis, ranking logic and time-series insights.

### 2. Integrated Sales Planning & Forecasting
Bottom-up planning model with data actions, multi-actions, allocation drivers, version comparison and forecasting.
